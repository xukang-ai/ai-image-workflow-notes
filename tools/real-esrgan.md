# Real-ESRGAN

## What it is
Real-ESRGAN is an AI-based image upscaling and restoration tool.
It is widely used for enhancing low-resolution images and old photos.

## Why I care
- Useful for restoring old photos
- Helpful for upscaling small images for print or stock
- Many AI image tools are based on this model

## Typical use cases
- Old photo restoration
- Image upscaling before PS retouch
- Improving AI-generated images

## Notes
- This is a backend tool, often used by other software
- I don't need to train models, only use pre-trained ones
