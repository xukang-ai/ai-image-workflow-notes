# AI Image Workflow Notes

> Personal notes for AI-assisted image editing, photography post-processing, and teaching.

---

## Why this repository exists

This repository is used to organize:
- AI image editing tools and notes
- Practical workflows for photo restoration and upscaling
- PS + AI teaching materials
- Personal experiments and observations

The goal is to build a long-term, reusable reference for both work and teaching.

Step-by-step workflows for common use cases:

- AI-assisted illustration

### 3. Teaching Notes
Explanations, examples, and materials used in classes.
